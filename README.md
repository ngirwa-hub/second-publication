This repository contains files that support the reproducibility of the experiments presented in our second manuscript. 

The folder structures are as follows: 
1. analysis folder: contains all the analysis scripts and the generated outputs that have been reported in the second manuscript
2. modefiles folder: contains the files for generating the LLM experts used in the prompts to produce the data analyzed in this manuscript
3. pythonscripts folder: contains the Python scripts used to prompt the LLM experts.
